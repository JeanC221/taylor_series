# Proyecto de Series de Taylor

Este proyecto es una aplicación web que resuelve y visualiza series de Taylor utilizando Python y Flask.

## Requisitos

- Python 3.12 o superior
- Flask 3.0.3
- SymPy
- Matplotlib
- NumPy
